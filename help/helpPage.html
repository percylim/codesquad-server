Getting start

  • Step 1: Register an Account with System include Company ID, Company Name, Admin Login ID and Password.
    After register success System will create a database for the registered Company.
  • Step 2: A confirmation email will send to user to request for payment. After payment a login password will be
    provided and user can change the password.
  • Step 3: After Register user will be able to login with admin ID and Password and allow to setup all account and profile, employee password then to start the account.
